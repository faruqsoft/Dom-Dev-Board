Question 1: What are the different ways to select an element in the DOM?

Answer:There are 5 ways in which i can select elements in a DOM using selectors.
1. getElementsByTagName();
2. getElementsByClassName();
3. getElementById();
4. querySelector();
5. querySelectorAll();

Question 2: What is the difference between innerHTML, innerText, and textContent?

Answer:
TextContents: is all text contained by an element and all its children that are for formatting purposes only.
InnerText: returns all text contained by an element and all its child elements.
InnerHtml: returns all text, including html tags, that is contained by an element.

Question 3: What is event delegation in the DOM?

Answer:
Event delegation in the DOM is a JavaScript technique where I attach a single event listener to a parent element, allowing it to handle events triggered on its child elements, instead of adding separate listeners to each child element; this leverages the "event bubbling" mechanism where events triggered on a child element propagate up to its ancestors in the DOM tree, making it efficient for handling events on large sets of dynamically added elements.

Question 4: What is event bubbling in the DOM?

Answer:
Event bubbling in the DOM is a process that propagates events up through the DOM tree, starting with the innermost element. This process is similar to ripples spreading out in a pond.
